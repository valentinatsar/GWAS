# Quality Control of 50K genotypes
PLINK: Whole genome data analysis toolset (https://zzz.bwh.harvard.edu/plink/)

Supports .ped/.map and binary .fam/.bed/.bim format files 

```bash
#Load the software
module load plink/v1.9

# Exclude indviduals with no available phenotypic information 
plink --bfile boutsko3 --chr-set 26 no-xy --remove boutsko_no-pheno.txt --make-bed --out boutsko_only-pheno

# Exclude remaining unmapped SNP markers 
plink --bfile boutsko_only-pheno --allow-extra-chr --chr-set 26 no-xy --exclude snps_bou_remove.txt --make-bed --out boutsko_only-mapped

#Perform Quality Control
plink --bfile boutsko_only-mapped --allow-extra-chr --chr-set 26 no-xy --geno 0.02 --maf 0.02 --make-bed --mind 0.05 --out boutsko_qc
